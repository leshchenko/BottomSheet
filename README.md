# Installation

## CocoaPods

Add next to your Podfile:
```
pod 'HorizontalCenteredCollectionView', '~> 1.0'
```

## Manually

To integrate HorizontalCenteredCollectionView manually into your project, copy files from `CollectionView` folder into your Xcode project.
